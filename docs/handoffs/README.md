# Handoffs

Ephemeral Dev / QA handoff files for the currently open work ID (or open baseline).

| Artifact | Filename (fixed) |
|----------|------------------|
| Dev handoff | `HANDOFF-DEV.md` |
| QA handoff | `HANDOFF-QA-pass1.md` / `HANDOFF-QA-pass2.md` |
| Baseline handoff | `HANDOFF-QA-baseline.md` |

**Work ID** stays in the file body. Filenames do not include feature slug or backlog ID.

**Next baseline ID** and lifespan rules: [`../reports/README.md`](../reports/README.md). Protocol: [`../protocols/HANDOFF.md`](../protocols/HANDOFF.md).
