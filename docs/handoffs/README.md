# Handoffs

Ephemeral Dev / QA handoff files for open work IDs (and open baselines).

| Artifact | Pattern |
|----------|---------|
| Dev handoff | `HANDOFF-DEV-{feature-slug}-{NNNN}.md` |
| QA handoff | `HANDOFF-QA-{feature-slug}-{NNNN}-pass1.md` (also `-pass2`) |
| Baseline handoff | `HANDOFF-QA-baseline-{NNNN}.md` |
